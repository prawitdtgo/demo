HTTP Status Code Error Code Recommendation
401 authorization_header_not_found Please send an Authorization header in the request.
invalid_token Please check if the access token scopes are valid.
expired_token Please refresh the access token.
403 unauthorized_access The client application should notify the end user that there were insufficient privileges to complete the operation.
404 item_not_found Please check the item identifier.
500 invalid_request * This error code will be presented if the client application is an SPA.
invalid_grant * Please check if the code, code_verifier, scopes, or refresh_token parameter is valid.
unauthorized_client * Please check if the client_id parameter is valid.
invalid_client * Please check if the client_secret or redirect_uri parameter is valid.
temporarily_unavailable * Retry the request after a small delay. The client application might explain to the end user that its response is delayed because the server is temporarily too busy to handle the request.
internal_server_error Retry the request after a small delay. After that, if it still does not work, please contact the system administrator.

Note: * means those error codes will only be presented in the following web services below.

  • POST https://web-services.dtgo.com/api/v1/oauth2/authorization-code-grant
  • POST https://web-services.dtgo.com/api/v1/oauth2/refresh-token-grant
  • POST https://web-services.dtgo.com/api/v1/oauth2/client-credentials-grant
